= time_of_day

Have you ever worked with ActiveRecord and SQL's time type? If you did, you know it sucks. Not anymore.

== Setup

In the `Gemfile` add:
  
  gem 'time_of_day', :git => 'git://github.com/lailsonbm/time_of_day'

Run `bundler`:

  bundle install

Just that.

== Compatibility
 
Tested with Rails 3.0.0.beta4 on Ruby 1.9.2-preview3 and Ruby 1.8.7.
